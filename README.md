# 🚀 MetalVision - AI Material Detection using YOLOv8

MetalVision is an AI-powered web application that detects whether a material is **Copper** or **Aluminum** from an uploaded image using a YOLOv8 model.

## 📌 Features

- 🔍 AI-based Copper & Aluminum Detection
- 📤 Image Upload
- 🤖 YOLOv8 Classification
- 📊 Confidence Score
- 📈 AI Detection Statistics
- 📝 Prediction History
- 📄 PDF Report Generation
- ⚡ FastAPI Backend
- 🎨 Modern React + Tailwind CSS UI
- 📱 Responsive Design

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/gaddebhagyaanjali/MetalVision.git
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 📸 Application Workflow

1. Upload a material image.
2. AI analyzes the image using YOLOv8.
3. Material prediction is displayed.
4. Confidence score is shown.
5. Prediction is saved to history.
6. Generate and download a PDF report.
